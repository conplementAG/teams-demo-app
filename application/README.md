# Example Application

## Introduction

There is an example application that consists of two components.

* Frontend: HTML-UI, Ajax Requests
* Backend: Data-Storage, REST-API

## Getting Started

### docker-compose

```bash
docker-compose up --build
```

Browse to http://localhost:8080
